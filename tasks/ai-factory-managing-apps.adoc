[#ai-factory-apps-install]
== How do I install AI applications?

. From the main left panel, select menu:SUSE AI Factory[Apps].
+
ifeval::["{PROF_PRODUCT}" == "saif"]
. Select btn:[SUSE AI Library] and select the application tile.
endif::[]
ifeval::["{PROF_PRODUCT}" == "saifnv"]
. Select btn:[NVIDIA Library] and select the application tile.
endif::[]
+
You can also limit the number of displayed applications by entering a search string.
. Use the `Basic Information` screen of the installation wizard to update the instance name, namespace, and version of the application.
+
[#fig-aif-apps-install-basic]
.Basic screen of the application installation wizard
image::aif-apps-install1.png[Screenshot showing basic information about the installed application,width=50%]
+
. On the `Target Cluster` screen, you can review hardware requirements for the workload, specify the deployment type, and which cluster to deploy the application to.
+
[#fig-aif-apps-install-target]
.Target Cluster screen of the application installation wizard
image::aif-apps-install2.png[Screenshot showing information about the installation target of an application,width=50%]
+
. Use the `Configuration` screen to change individual settings in the {helm} chart of the application.
+
[#fig-aif-apps-install-config]
.Configuration screen of the application installation wizard
image::aif-apps-install3.png[Screenshot showing information about application configuration,width=50%]
+
. The `Review` screen summarizes the basic information about the deployment.
Start the deployment by selecting the btn:[Install] button.
+
[#fig-aif-apps-install-review]
.Configuration screen of the application installation wizard
image::aif-apps-install4.png[Screenshot reviewing the installation,width=50%]
+
. After the installation progress bar reaches 100%, you can check the workload status by selecting menu:SUSE AI Factory[Workloads] from the left menu.

== How do I manage application instances?

[#fig-aif-apps-manage]
.List of AI workloads to manage
image::aif-apps-manage.png[Screenshot showing a list of AI workloads to manage,scaledwidth=100%]

. From the main left panel, select menu:SUSE AI Factory[Workloads].
. Select the btn:[Manage] button pertaining to the application you want to manage.
. In the `Manage` wizard, you can update the application version, the destination cluster, and the details of the {helm} chart.
. Confirm with btn:[Save].


[#ai-factory-apps-delete]
== How do I delete application instances?

. From the main left panel, select menu:SUSE AI Factory[Workloads].
. Select the btn:[Delete] button pertaining to the application you want to uninstall, and confirm the deletion.
