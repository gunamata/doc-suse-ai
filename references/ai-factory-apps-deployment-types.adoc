*{fleeta} Bundle*::
Deploy the application directly as a {fleeta} bundle managed by {rancherfleet}.
This option is ideal if you want to use {fleeta} without maintaining a separate Git repository. For more details, see the link:https://fleet.rancher.io/0.12/[Rancher Fleet documentation].

*Publish to {fleeta} Git*::
Publish the application configuration to an external Git repository monitored by {fleeta}.
{fleeta} automatically deploys, monitors, and reconciles the application using a GitOps workflow.

*{helm}*::
Deploy the application using {helm} commands or the {ranchera} Apps UI.
Use this method for direct installation and manual lifecycle management.
+
[NOTE]
====
For specific large blueprints, {helm} deployment options are unavailable and are disabled in the {ranchera} user interface.
====
